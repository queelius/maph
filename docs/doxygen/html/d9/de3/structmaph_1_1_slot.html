Detailed Description

Storage slot for key-value pairs.

Each slot is 512 bytes and cache-line aligned (64 bytes) for optimal performance. The slot contains metadata and the actual value data.

Memory Layout:

  • 8 bytes: atomic hash_version (32-bit hash + 32-bit version)
  • 4 bytes: data size
  • 4 bytes: reserved for future use
  • 496 bytes: value data
