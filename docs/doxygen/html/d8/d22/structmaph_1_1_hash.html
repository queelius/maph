Detailed Description

Hash function implementation for key distribution.

Uses FNV-1a hash algorithm for good distribution and speed. Supports both scalar and SIMD (AVX2) implementations for batch operations.

Compute hash and slot index for a key.

Uses FNV-1a algorithm:

  1. Start with offset basis (2166136261)
  2. For each byte: XOR with hash, multiply by FNV prime (16777619)
  3. Never returns 0 (reserved for empty slots)
Parameters
keyKey to hash
num_slotsTotal number of slots (for modulo operation)
Returns
Hash result containing full hash and slot index
