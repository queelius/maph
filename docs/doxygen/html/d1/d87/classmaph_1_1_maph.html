Detailed Description

Main maph class - lean and fast.

High-performance memory-mapped key-value store.

Provides a persistent key-value database with sub-microsecond lookups using memory-mapped I/O and approximate perfect hashing.

Features:

  • O(1) average-case lookups
  • Zero-copy string operations
  • Lock-free reads
  • Automatic persistence via mmap
  • Parallel batch operations
  • Optional async durability

Usage Example:

auto db = Maph::create("data.maph", 1000000);
db->set("user:123", "{\"name\":\"Alice\"}");
auto value = db->get("user:123");
