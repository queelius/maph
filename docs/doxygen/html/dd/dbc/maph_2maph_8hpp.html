Detailed Description

Lean, mean, mmap-based perfect hash JSON mapping.

Philosophy: Speed at every decision point

  • mmap for zero-copy access
  • Fixed slots for O(1) everything
  • Perfect hash for direct indexing
  • JSON for flexibility (only overhead we accept)

struct maph::Header

Database file header containing metadata.

The header is exactly 512 bytes and contains database configuration and runtime statistics. It's placed at the beginning of the mmap'd file.
