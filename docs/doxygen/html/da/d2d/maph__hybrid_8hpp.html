Detailed Description

Hybrid static/dynamic hash with perfect hash for hot keys.

Key insight: Store (key_hash, value) instead of just value

  • Detect empty slots (hash == 0)
  • Add new keys dynamically with probing
  • Optional expiration timestamps
  • Perfect hash for hot path, dynamic for long tail
